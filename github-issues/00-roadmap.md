# Enhancement: Feature Improvement Roadmap

**Label:** `enhancement`

---

このissueは、MIB Browserの機能改善を追跡するためのメインissueです。

## 実装予定の改善項目

### 優先度: 高（すぐに実装可能）

- [ ] コピー機能 - OIDや詳細情報のクリップボードコピー
- [ ] トースト通知システム - アップロード成功/失敗の視覚的フィードバック
- [ ] 検索結果の改善 - 件数表示、ナビゲーション、ハイライト強化
- [ ] OIDパスBreadcrumb - クリック可能な階層パス表示
- [ ] ソート機能 - ファイルリスト・ツリーのソート

### 優先度: 中

- [ ] ナビゲーション履歴 - 戻る/進むボタン
- [ ] お気に入り/ブックマーク - よく使うノードの保存
- [ ] キーボードショートカット - Ctrl+Fなど
- [ ] ダークモード - テーマ切り替え
- [ ] MIB名の表示改善 - MODULE名の表示
- [ ] 正規表現検索 - 高度な検索機能
- [ ] URLパラメータ対応 - 特定ノードへの直リンク

### 優先度: 低（長期的な機能）

- [ ] OID変換ツール - 数値OID ⇔ テキストOID変換
- [ ] MIB比較機能 - 2ファイルの差分表示
- [ ] 依存関係可視化 - IMPORTSの解析とグラフ表示
- [ ] 仮想スクロール - 大規模ツリーのパフォーマンス改善
- [ ] PDFエクスポート - レポート生成機能
- [ ] テーブル構造検出 - SNMPテーブルの自動識別

## 実装の進め方

1. 優先度の高い機能から順に実装
2. 各機能は個別issueで管理
3. ユーザーフィードバックに基づいて優先度を調整
